Habilidad de Analizador Dinámico para OpenClaw.

## Qué Resuelve
Analizador impulsado por IA para tareas de análisis.

## Características
- Activación automática cuando se detectan tareas relevantes
- Resultados profesionales y listos para producción
- Enfoque prioritario en la seguridad
- Soporte para reversión

## Uso
/dynamic-analyzer

## Ejemplo
```
Analizar código en busca de problemas de análisis.
```

## Instalación
Esta habilidad está disponible automáticamente en OpenClaw.
